# Microprocesadores y Acondicionadores de Señales

Este repositorio contiene material de estudio y ejercicios para la asignatura de Microprocesadores y Acondicionadores de Señales.

## Estructura del Repositorio

*   `Apuntes/`: Contiene apuntes de clase organizados por temas.
*   `Ejercicios/`: Contiene ejercicios prácticos para afianzar los conocimientos.
*   `Examenes/`: Contiene exámenes de años anteriores.
*   `Practica1/`: Contiene la práctica 1 de la asignatura.
*   `Teoria/`: Contiene los temas de la asignatura en formato PDF.
